# Triage Labels

This repository uses the default canonical triage label vocabulary:

- `needs-triage` - maintainer needs to evaluate the issue
- `needs-info` - waiting on reporter input
- `ready-for-agent` - sufficiently specified for an AFK agent
- `ready-for-human` - needs human implementation
- `wontfix` - will not be actioned

Use these exact strings when triaging. Do not create alternate spellings or duplicate labels unless the repository's issue tracker is intentionally migrated to a different convention.
